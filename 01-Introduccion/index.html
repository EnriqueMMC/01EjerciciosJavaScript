JavaScript

Formas de añadir código de JavaScript en html

  • Entre las etiquetas head, dentro de las etiquetas <script></script>
  • Entre las etiquetas body, en cualquier lugar aunque se suele poner al final, antes de cerrar el body
  • En un archivo externo.

¿Qué forma es la más recomendada?

  • 📌 Enlazar el código JavaScript desde un archivo externo es la opción más recomendada
  • 📌 Esto nos permite utilizar el mismo código desde diferentes archivos html
  • 📌 Para mantener nuestro proyecto organizado se recomienda crear un directorio para los archivos .js
  • 📌 Para crear el enlace entre el archivo .js y nuestro .html la sintaxis es: <script src="ruta relativa al archivo"></script>
  • 📌 Es importante poner la ruta relativa para que al subir el proyecto al servidor no hay que cambiar nada

¿Quien se encarga de interpretar y ejecutar el código de JavaScript?

  • En la actualidad todos los navegadores son capaces de interpretar y ejecutar JavaScript

¿Qué podemos hacer con JavaScript?

  • Validar formularios
  • Crear animaciones
  • Crear aplicaciones web
  • Crear juegos
  • Crear aplicaciones de escritorio
