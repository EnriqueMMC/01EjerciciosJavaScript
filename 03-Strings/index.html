Strings o cadenas de texto

Las cadenas de texto son un tipo de dato que se utiliza para representar texto en programación. En JavaScript, las cadenas de texto se pueden crear utilizando comillas simples (' '), comillas dobles (" ") o comillas invertidas (` `).

Las cadenas de texto son inmutables, lo que significa que no se pueden cambiar una vez que se han creado. Sin embargo, se pueden crear nuevas cadenas de texto a partir de cadenas existentes utilizando métodos y operadores.

Las cadenas de texto son un tipo de dato primitivo en JavaScript y se pueden utilizar para representar cualquier tipo de información textual, como nombres, direcciones, mensajes, etc.

En JavaScript, las cadenas de texto son objetos y tienen propiedades y métodos que se pueden utilizar para manipularlas. Algunos de los métodos más comunes son:

  • length: Devuelve la longitud de la cadena de texto. 
    let texto = "Hola Mundo"; console.log(texto.length); // 10
  • toUpperCase(): Convierte la cadena de texto a mayúsculas. 
    let texto = "Hola Mundo"; console.log(texto.toUpperCase()); // "HOLA MUNDO"
  • toLowerCase(): Convierte la cadena de texto a minúsculas. 
    let texto = "Hola Mundo"; console.log(texto.toLowerCase()); // "hola mundo"
  • charAt(): Devuelve el carácter en una posición específica de la cadena de texto. 
    let texto = "Hola Mundo"; console.log(texto.charAt(4)); // " "
  • indexOf(): Devuelve la posición de la primera aparición de un carácter o una subcadena en la cadena de texto. 
    let texto = "Hola Mundo"; console.log(texto.indexOf("Mundo")); // 5
  • substring(): Devuelve una subcadena de la cadena de texto. 
    let texto = "Hola Mundo"; console.log(texto.substring(0, 4)); // "Hola"
  • split(): Divide la cadena de texto en un array de subcadenas. 
    let texto = "Hola Mundo"; console.log(texto.split(" ")); // ["Hola", "Mundo"]
  • replace(): Reemplaza una parte de la cadena de texto por otra. 
    let texto = "Hola Mundo"; console.log(texto.replace("Mundo", "JavaScript")); // "Hola JavaScript"
  • trim(): Elimina los espacios en blanco al principio y al final de la cadena de texto. 
    let texto = "   Hola Mundo   "; console.log(texto.trim()); // "Hola Mundo"
  • includes(): Comprueba si una cadena de texto contiene otra cadena de texto. 
    let texto = "Hola Mundo"; console.log(texto.includes("Mundo")); // true
  • startsWith(): Comprueba si una cadena de texto comienza con otra cadena de texto. 
    let texto = "Hola Mundo"; console.log(texto.startsWith("Hola")); // true
  • endsWith(): Comprueba si una cadena de texto termina con otra cadena de texto. 
    let texto = "Hola Mundo"; console.log(texto.endsWith("Mundo")); // true
  • repeat(): Repite la cadena de texto un número específico de veces. 
    let texto = "Hola "; console.log(texto.repeat(3)); // "Hola Hola Hola "
  • concat(): Concatena dos o más cadenas de texto. 
    let texto1 = "Hola"; let texto2 = "Mundo"; console.log(texto1.concat(" ", texto2)); // "Hola Mundo"
  • padStart(): Añade caracteres al principio de la cadena de texto hasta alcanzar una longitud específica. 
    let texto = "5"; console.log(texto.padStart(3, "0")); // "005"
  • padEnd(): Añade caracteres al final de la cadena de texto hasta alcanzar una longitud específica. 
    let texto = "5"; console.log(texto.padEnd(3, "0")); // "500"
