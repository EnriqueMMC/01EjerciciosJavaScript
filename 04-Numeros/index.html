Números

Los números son un tipo de dato que se utiliza para representar valores numéricos en programación. En JavaScript, los números pueden ser enteros o decimales y se pueden utilizar para realizar operaciones matemáticas.

Los números en JavaScript son de precisión doble, lo que significa que pueden representar números muy grandes o muy pequeños con una precisión de hasta 15 dígitos decimales.

Los números son un tipo de dato primitivo en JavaScript y se pueden utilizar para representar cualquier tipo de información numérica, como cantidades, precios, edades, etc.

En JavaScript, los números son objetos y tienen propiedades y métodos que se pueden utilizar para manipularlos. Algunos de los métodos más comunes son:

  • parseInt(): Convierte una cadena de texto en un número entero. 
    let numero = "10"; console.log(parseInt(numero)); // 10
  • parseFloat(): Convierte una cadena de texto en un número decimal. 
    let numero = "10.5"; console.log(parseFloat(numero)); // 10.5
  • toFixed(): Formatea un número con un número específico de decimales. 
    let numero = 10.123456; console.log(numero.toFixed(2)); // 10.12
  • Math.abs(): Devuelve el valor absoluto de un número. 
    let numero = -10; console.log(Math.abs(numero)); // 10
  • Math.round(): Redondea un número al entero más cercano. 
    let numero = 10.5; console.log(Math.round(numero)); // 11
  • Math.floor(): Redondea un número hacia abajo al entero más cercano. 
    let numero = 10.9; console.log(Math.floor(numero)); // 10
  • Math.ceil(): Redondea un número hacia arriba al entero más cercano. 
    let numero = 10.1; console.log(Math.ceil(numero)); // 11
  • Math.max(): Devuelve el número más grande de una lista de números. 
    let numeros = [1, 2, 3, 4, 5]; console.log(Math.max(...numeros)); // 5
  • Math.min(): Devuelve el número más pequeño de una lista de números. 
    let numeros = [1, 2, 3, 4, 5]; console.log(Math.min(...numeros)); // 1
  • Math.random(): Devuelve un número aleatorio entre 0 y 1. 
    console.log(Math.random()); // 0.123456789
  • Math.sqrt(): Devuelve la raíz cuadrada de un número. 
    let numero = 16; console.log(Math.sqrt(numero)); // 4
  • Math.pow(): Devuelve la potencia de un número. 
    let base = 2; let exponente = 3; console.log(Math.pow(base, exponente)); // 8
