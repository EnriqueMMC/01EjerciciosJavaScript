Arrays

Un array es una estructura de datos que permite almacenar múltiples valores en una sola variable. En JavaScript, los arrays son objetos especiales que pueden contener cualquier tipo de dato, incluyendo otros arrays.

Los arrays son útiles para almacenar colecciones de datos y se pueden manipular fácilmente utilizando métodos integrados.

Ejemplo de un array:

            
                let frutas = ['manzana', 'banana', 'naranja'];
                console.log(frutas[0]); // salida: 'manzana'

Los arrays en JavaScript son dinámicos, lo que significa que pueden crecer y decrecer según sea necesario. Puedes agregar o eliminar elementos de un array utilizando métodos como push(), pop(), shift(), y unshift().

Ejemplo de agregar y eliminar elementos:

            
            let frutas = ['manzana', 'banana', 'naranja'];
            frutas.push('uva'); // agrega 'uva' al final del array
            console.log(frutas); // salida: ['manzana', 'banana', 'naranja', 'uva']
                
            frutas.pop(); // elimina el último elemento del array
            console.log(frutas); // salida: ['manzana', 'banana', 'naranja']

También puedes recorrer un array utilizando un bucle for o el método forEach().

Ejemplo de recorrer un array:

            
            let frutas = ['manzana', 'banana', 'naranja'];
            frutas.forEach(function(fruta) {
                console.log(fruta);
            });

En este ejemplo, el método forEach() recorre cada elemento del array y ejecuta la función proporcionada para cada uno de ellos.

Ejemplo de recorrer un array utilizando un bucle for:

            
            let frutas = ['manzana', 'banana', 'naranja'];
            for (let i = 0; i < frutas.length; i++) {
                console.log(frutas[i]);
            }

Los arrays también tienen propiedades útiles, como length, que devuelve la cantidad de elementos en el array.

Ejemplo de usar la propiedad length:

            
                let frutas = ['manzana', 'banana', 'naranja'];
                console.log(frutas.length); // salida: 3
