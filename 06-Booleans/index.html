Booleanos

En JavaScript, los booleanos son un tipo de dato que solo puede tener uno de dos valores: true o false. Estos valores se utilizan comúnmente en estructuras de control, comparaciones y lógica de programación.

  • true: Representa un valor verdadero.
  • false: Representa un valor falso.

Por ejemplo:

            
            const esMayor = 10 > 5; // true
            const esIgual = 10 === '10'; // false
