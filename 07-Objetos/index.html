Objetos

Los objetos son una colección de propiedades y métodos que representan un conjunto de datos relacionados. En JavaScript, los objetos son un tipo de dato complejo que se utiliza para almacenar información en forma de pares clave-valor.

Los objetos pueden contener diferentes tipos de datos, como números, cadenas de texto, booleanos, arreglos y otros objetos. Los objetos son una parte fundamental de la programación orientada a objetos y se utilizan para modelar entidades del mundo real.

En JavaScript, los objetos se pueden crear de varias maneras, como utilizando la sintaxis de objeto literal, la función constructora o la clase. Algunos de los métodos más comunes para trabajar con objetos son:

  • Crear un objeto: Se puede crear un objeto utilizando la sintaxis de objeto literal. 
    let persona = { nombre: "Juan", edad: 30 };
  • Acceder a propiedades: Se puede acceder a las propiedades de un objeto utilizando la notación de punto o la notación de corchetes. 
    console.log(persona.nombre); // Juan
  • Modificar propiedades: Se pueden modificar las propiedades de un objeto asignando un nuevo valor. 
    persona.edad = 31;
  • Añadir propiedades: Se pueden añadir nuevas propiedades a un objeto asignando un valor a una nueva clave. 
    persona.direccion = "Calle Falsa 123";
  • Eliminar propiedades: Se pueden eliminar propiedades de un objeto utilizando el operador delete. 
    delete persona.edad;
  • Recorrer un objeto: Se puede recorrer un objeto utilizando un bucle for...in. 
    for (let clave in persona) { console.log(clave + ": " + persona[clave]); }
  • Convertir un objeto a JSON: Se puede convertir un objeto a una cadena JSON utilizando el método JSON.stringify(). 
    let json = JSON.stringify(persona);
  • Convertir JSON a un objeto: Se puede convertir una cadena JSON a un objeto utilizando el método JSON.parse(). 
    let objeto = JSON.parse(json);
  • Clonar un objeto: Se puede clonar un objeto utilizando el operador de propagación o el método Object.assign(). 
    let clon = { ...persona };
  • Unir dos objetos: Se pueden unir dos objetos utilizando el operador de propagación o el método Object.assign(). 
    
            // Usando el operador de propagación
            let objeto1 = { a: 1, b: 2 };
            let objeto2 = { c: 3, d: 4 };
            let combinado = { ...objeto1, ...objeto2 };

            // Usando Object.assign()
            let combinado2 = Object.assign({}, objeto1, objeto2);
  • Desestructuración de objetos: Se puede desestructurar un objeto para extraer propiedades en variables individuales. 
    let { nombre, edad } = persona;
  • Uso de métodos: Se pueden definir métodos dentro de un objeto. 
    let persona = { nombre: "Juan", saludar: function() { console.log("Hola, " + this.nombre); } };
  • Uso de this: Se puede utilizar la palabra clave this para referirse al objeto actual dentro de un método. 
    persona.saludar(); // Hola, Juan
  • Uso de Object.keys(): Se puede utilizar el método Object.keys() para obtener un array con las claves de un objeto. 
    let claves = Object.keys(persona);
  • Uso de Object.values(): Se puede utilizar el método Object.values() para obtener un array con los valores de un objeto. 
    let valores = Object.values(persona);
  • Uso de Object.entries(): Se puede utilizar el método Object.entries() para obtener un array con los pares clave-valor de un objeto. 
    let entradas = Object.entries(persona);
  • Uso de Object.freeze(): Se puede utilizar el método Object.freeze() para evitar que un objeto sea modificado. 
    Object.freeze(persona);
  • Uso de Object.seal(): Se puede utilizar el método Object.seal() para evitar que se añadan o eliminen propiedades de un objeto, pero se pueden modificar las propiedades existentes. 
    Object.seal(persona);
  • Uso de Object.assign(): Se puede utilizar el método Object.assign() para copiar propiedades de un objeto a otro. 
    let nuevoObjeto = Object.assign({}, persona);
  • Uso de Object.create(): Se puede utilizar el método Object.create() para crear un nuevo objeto con un prototipo específico. 
    let nuevoObjeto = Object.create(persona);
  • Objetos dentro de objetos: Un objeto puede contener otros objetos como propiedades. 
    let empresa = { 
                nombre: "TechCorp", 
                direccion: { 
                    calle: "Avenida Principal", 
                    numero: 123, 
                    ciudad: "Ciudad Tech" 
                } 
            };
  • Desestructuración de objetos anidados: Se puede desestructurar un objeto anidado para extraer propiedades específicas. 
    let { direccion: { calle, ciudad } } = empresa;
            console.log(calle); // Avenida Principal
            console.log(ciudad); // Ciudad Tech
  • Creación de objetos con Object Literal: Se puede crear un objeto utilizando la sintaxis de objeto literal. 
    let auto = { marca: "Toyota", modelo: "Corolla", año: 2022 };
  • Creación de objetos con Object Constructor: Se puede crear un objeto utilizando un constructor. 
    function Auto(marca, modelo, año) {
                this.marca = marca;
                this.modelo = modelo;
                this.año = año;
            }
            let miAuto = new Auto("Honda", "Civic", 2023);
