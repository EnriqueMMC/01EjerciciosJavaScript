Operadores

Los operadores son símbolos que se utilizan para realizar operaciones en JavaScript. Existen diferentes tipos de operadores, como los aritméticos, de comparación, lógicos, de asignación, entre otros.

Los operadores aritméticos son los más comunes y se utilizan para realizar operaciones matemáticas básicas, como la suma, resta, multiplicación y división. Los operadores de comparación se utilizan para comparar valores y devolver un valor booleano (true o false) según el resultado de la comparación.

Los operadores lógicos se utilizan para combinar expresiones booleanas y devolver un valor booleano. Los operadores de asignación se utilizan para asignar valores a variables.

Los operadores son una parte fundamental de la programación y se utilizan en casi todas las aplicaciones y scripts de JavaScript.

Algunos de los operadores más comunes son:

  • Operadores Aritméticos: Se utilizan para realizar operaciones matemáticas. 
    let suma = 5 + 3; // 8
  • Operadores de Comparación: Se utilizan para comparar valores. 
    let esIgual = (5 == 5); // true
  • Operadores Lógicos: Se utilizan para combinar expresiones booleanas. 
    let resultado = (true && false); // false
  • Operadores de Asignación: Se utilizan para asignar valores a variables. 
    let x = 10; // Asignación simple
  • Operadores de Incremento y Decremento: Se utilizan para aumentar o disminuir el valor de una variable en 1. 
    let x = 5; x++; // Incremento (x = 6)
  • Operadores Ternarios: Se utilizan para realizar una evaluación condicional en una sola línea. 
    let resultado = (5 > 3) ? "Mayor" : "Menor"; // "Mayor"
  • Operador de Coalescencia Nula: Se utiliza para asignar un valor por defecto si el valor es null o undefined. 
    let x = null; let y = x ?? 10; // y = 10
  • Operador de Propagación: Se utiliza para expandir elementos de un iterable en lugares donde se esperan cero o más argumentos (en llamadas a funciones) o elementos (en literales de matriz). 
    let arr1 = [1, 2]; let arr2 = [3, 4]; let arr3 = [...arr1, ...arr2]; // arr3 = [1, 2, 3, 4]
  • Operador de Asignación por Desestructuración: Se utiliza para extraer valores de arreglos o propiedades de objetos y asignarlos a variables. 
    let arr = [1, 2, 3]; let [a, b] = arr; // a = 1, b = 2
  • Operador de Comparación Estricta: Se utiliza para comparar valores y tipos de datos. 
    let esIgual = (5 === "5"); // false
  • Operador de Comparación No Estricta: Se utiliza para comparar valores sin tener en cuenta el tipo de dato. 
    let esIgual = (5 == "5"); // true
